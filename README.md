# 📸 LensCraft - Image Gallery

A modern, responsive image gallery with filtering, lightbox, and smooth animations built with HTML, CSS, and JavaScript.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)

## ✨ Features

- **Responsive Grid Layout** - Adapts beautifully to all screen sizes
- **Category Filtering** - Filter images by Nature, Urban, or Portrait
- **Grid/List View Toggle** - Switch between grid and list views
- **Lightbox Gallery** - Full-screen image preview with navigation
- **Smooth Animations** - Hover effects and transition animations
- **Lazy Loading** - Optimized image loading for better performance
- **Keyboard Navigation** - Use arrow keys and ESC in lightbox
- **Dark Theme** - Modern dark UI with gradient accents

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required

### Installation

1. Clone or download this repository:
```bash
git clone https://github.com/yourusername/ImageGallery-LensCraft.git
cd ImageGallery-LensCraft
```

2. Add your images to the `assets/images/` folder:
   - photo1.jpg - photo6.jpg (or rename your images accordingly)

3. Open `index.html` in your browser:
```bash
# Option 1: Double-click index.html
# Option 2: Use a local server
python -m http.server 8000
# Then visit http://localhost:8000
```
